# HikuCraft

## Commands List

### Home
to set a home somewhere <br />
`/sethome <name>` <br />
to show homes <br />
`/home` <br />
to teleport to a home <br />
`/home <name>` <br />
to delete a home <br />
`/delhome <name>` <br />

### Tp
to teleport to someone <br />
`/tp <username>` <br />
to accept a teleport request from someone <br />
`/tpaccept <username>` <br />
